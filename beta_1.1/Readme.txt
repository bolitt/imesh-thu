点云构建三角网格模型――说明文件（2010/12/26）


操作流程：
1. 解压缩Models.rar

2. 双击打开IMesh_beta1.1.exe

3. 文件 -> 打开 -> (读取特定模型)

4. 操作 -> 单步：单步运行演示
           暂停：暂停步骤演示
           步骤演示：显示每一步的构造过程
           运行至完成：显示最后结果

5. 其中提示信息在"输出"的"生成"窗口可以看到。

6. 关于用户浏览：
      鼠标拖拽：
          向上/下：向上或向下转动
          向左/右：向左或向右转动
          中间前进/后退：放大/缩小（Zoom in/out）

7. 设置 -> 显示设置：
       显示表面： 渲染表面三角形
       显示网格： 显示三角形网格
       显示法线： 显示每个点的法向量
       显示筛选点： 显示每个点是否参与构造（被忽略的点为红色）
            
！注：对于新的模型，需要先文件 -> 新建，然后再打开模型文件
